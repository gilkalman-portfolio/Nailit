# NailIt (MVP) – Gate 1

אפליקציית iOS עצמאית לטכנאיות ציפורניים, ללא שרת וללא סנכרון ענן. הנתונים נשמרים מקומית ב־SQLite בלבד, והסנכרון הוא חד־כיווני אל Apple Calendar.

## הרצה מקומית

```bash
npm install
npm run ios
```

> הפרויקט נבנה עם Expo SDK 50+ ותומך ב־iOS בלבד.

## הרשאות נדרשות

- יומן (Calendar) – ליצירה ועדכון אירועים.
- התראות (Notifications) – להתראות מלאי מקומיות ומיידיות בלבד.

## Gate 1 – מה כלול

- בסיס נתונים מקומי (SQLite) עם יצירת טבלאות וזריעת ערכי ברירת מחדל.
- דשבורד תורים יומיים עם כפתור ✅ לתורים שעברו ועדיין לא בוצעו.
- יצירת תור + יצירת אירוע ביומן של אפל (חד־כיווני בלבד).
- בדיקת חפיפות מול ה־DB המקומי.
- מסך הגדרות: ספים למלאי, לינק אינסטגרם, בחירת יומן ברירת מחדל, בדיקת התראות.
- התראות מיידיות לפי ספירה של תורים שבוצעו (ללא מודולו).

## מגבלות ידועות (Gate 1)

- אין סנכרון דו־כיווני מול Apple Calendar.
- אין מדיה/תמונות.
- אין אינטגרציה ל־WhatsApp.
- אין ייבוא מאנשי קשר.
- אין פרופיל לקוחה מלא מעבר למה שנדרש לדשבורד.

## שינוי שובר תאימות ב־Gate 1

- החל מהגרסה הזו, `start_time` נשמר כ־epoch milliseconds (מספר) במקום ISO string. כדי לפשט את המעבר, טבלת `Appointments` נמחקת ונוצרת מחדש בהפעלה הראשונה לאחר העדכון, ולכן תורים קיימים יימחקו. זה תקין ל־MVP מקומי בלבד.

## מבנה בסיסי

- `src/db` – שכבת SQLite והלוגיקה העסקית.
- `src/lib` – אינטגרציות יומן והתראות.
- `src/screens` – מסכים בסיסיים (דשבורד, יצירת תור, הגדרות).
